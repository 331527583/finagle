FixedNumReadsTransport

Related Doc: package benchmark

class FixedNumReadsTransport[In, Out] extends Transport[In, Out]

A Transport that allows a fixed number of reads. The output value of a read is a constant provided by readVal. Writes to the transport are simply dropped.
