com.twitter.finagle.toggle.flag

overrides

Related Doc: package flag

object overrides extends GlobalFlag[Map[String, Double]]

GlobalFlag source for the Flag-based ToggleMap.

Usage is of the form:

-com.twitter.finagle.toggle.flag.overrides=com.twitter.finagle.NewThing=0.1,com.twitter.finagle.OtherThing=0.99

Methods overrides$.let(id:String,fraction:Double* and overrides.letClear are available to help writing unit tests that manipulate flag-based Toggle values.

See also
