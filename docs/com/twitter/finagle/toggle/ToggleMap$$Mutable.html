com.twitter.finagle.toggle.ToggleMap

Mutable

Related Doc: package ToggleMap

trait Mutable extends ToggleMap

The ToggleMap interface is read only and this is the mutable side of it.

Implementations are expected to be thread-safe.

    iterator includes metadata from both self and that, with self's metadata taking precedence on conflicting ids.
