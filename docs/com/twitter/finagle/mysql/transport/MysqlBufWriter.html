  16. def owned(): Buf

    In keeping with Buf terminology, creates a potential zero-copy Buf which has a reference to the same region as the BufWriter.

    In keeping with Buf terminology, creates a potential zero-copy Buf which has a reference to the same region as the BufWriter. That is, if any methods called on the builder are propagated to the returned Buf. By definition, the builder owns the region it is writing to so this is the natural way to coerce a builder to a Buf.
